    INTRODUCTION

    The program searches and reports whether a given number is present in the array or not.

    WORKING

    The program first takes in the number of elements to be input.

    It then takes in the respective elements in random order.

    It then sorts these elements in ascending order.

    It then takes in the element to be searched.

    It then searches for the element in the array using binary search and displays the results.
